Laparoscopic Device: Kinematics & Force Simulation 🏥⚙️
📌 Project Overview

This project details the mechanical design and dynamic simulation of a specialized laparoscopic surgical tool. The primary engineering objective was to design a reliable mechanical actuation system and accurately model the resistance and force response of the tool's internal mechanisms during operation.
🛠️ Mechanical Design & Mechanisms

The device utilizes precision mechanisms to translate user input into accurate end-effector actuation.

    Native CAD: Designed parametrically in SolidWorks. The repository includes native .SLDPRT and universally accessible .STEP files.

    Cam Barrel Mechanism: Engineered to convert rotary/linear input into specific stroke lengths required for surgical precision.

    Trigger Release Technique: Designed a mechanical release system that ensures consistent actuation and safe reset functionality.

🧮 Dynamic Simulation & Force Modeling

A major component of this project was validating mechanical forces before physical prototyping. The entire kinematic chain was modeled and analyzed using MATLAB & Simscape Multibody.

    Resistance Simulation: Modeled the mechanical resistance of the trigger mechanism to ensure ergonomic operation.

    Mechanical Modeling: The simulation accounts for spring rates, dampening, and the mechanical advantage generated by the cam profile geometry.

📂 Repository Structure

    /cad — Contains SolidWorks feature tree files (.SLDPRT, .SLDASM) and .STEP files.

    /sim — Contains MATLAB/Simscape Multibody files (.slx) and initialization scripts.

    /docs — Contains mechanism diagrams and mathematical derivations for the cam profile.

🚀 How to Explore this Project

    CAD Review: Open the .STEP files in any standard CAD package or use SolidWorks to view the native feature trees.

    Simulation: To run the dynamic simulation, open the .slx file in MATLAB and execute the setup script to load the mechanical parameters.
